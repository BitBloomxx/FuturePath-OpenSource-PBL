 * TRAINING AND PLACEMENT (T&P) CELL MANAGEMENT SYSTEM
 
 DESCRIPTION:
 * This is a comprehensive matching engine designed for university placement cells.
 * It evaluates students based on academic performance (SGPA), technical skills,and branch eligibility, then matches them against top-tier corporate requirements.
 DESIGN PATTERNS USED:
 - Object-Oriented Programming (Classes & Inheritance)
 - Encapsulation (Private/Protected data members)
 - Polymorphism (Virtual functions for company matching)
 - Container-Based Logic (Vectors instead of raw arrays)
 KEY FEATURES:
 1. Branch-Wise Filtering: Ensures students only apply for roles open to their stream.
 2. Skill Gap Analysis: Tells students exactly how much they need to improve.
 3. Package Prediction: Calculates potential salary based on a proprietary score.
 4. Company Rating: Each company and role is ranked on a scale of 1-10.
